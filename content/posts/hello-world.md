# Hello World

欢迎来到我的个人博客！

这是使用 **React + Next.js + GitHub Pages** 构建的第一篇文章。

## 关于这个项目

这个博客项目采用了现代化的前端技术栈：

- **Next.js 14**：使用 App Router
- **Static Site Generation (SSG)**：静态生成，性能优秀
- **Markdown**：使用 Markdown 编写文章
- **GitHub Pages**：免费托管
- **GitHub Actions**：自动化部署

## 技术特点

1. **纯静态站点**：无需后端服务
2. **SEO 友好**：HTML 预渲染
3. **快速加载**：静态资源优化
4. **自动部署**：推送到 main 分支自动发布
