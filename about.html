关于我

你好！

欢迎来到我的博客。我是一名热爱技术的开发者， 主要关注前端开发、Web 技术和软件工程实践。

这个博客使用以下技术栈构建：

  • Next.js 14（React 框架）
  • TypeScript
  • Tailwind CSS
  • 静态站点生成（SSG）
  • GitHub Pages 部署
  • GitHub Actions 自动化 CI/CD

为什么创建这个博客？

创建这个博客的目的是：

  • 记录学习过程中的心得体会
  • 分享技术解决方案和最佳实践
  • 构建个人知识体系
  • 与其他开发者交流学习

技术栈说明

本博客采用静态站点生成（SSG）方式，这意味着：

  • 所有页面在构建时生成
  • 无需运行时后端服务
  • 加载速度快，SEO 友好
  • 可以部署到静态托管服务（如 GitHub Pages）

博客内容以 Markdown 格式存储在 content/posts 目录下， 使用 Next.js App Router 实现页面路由。
